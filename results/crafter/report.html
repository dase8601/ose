A-B-M Experiment: Autonomous vs Fixed-Schedule Mode Switching

Empirical test of the Dupoux/LeCun/Malik A-B-M architecture (arXiv 2603.15381) on Crafter. System A = LeWM (~1.5M params, trains from pixels). System B = LSTM-PPO. System M = autonomous plateau-detect FSM vs fixed-schedule timer.
